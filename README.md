
  1️⃣ What is the difference between var, let, and const?
  Ans :
  1.var - old way. You can change it and declare it again. Not used much now.
  2.let - you can change the value, but cannot declare it again in the same place.
  3.const - cannot change the value after you set it.

  2️⃣ What is the spread operator (...)?
    Ans : the spread operator (...) takes a group (like an array) and turns it into individual items.


  3️⃣ What is the difference between map(), filter(), and forEach()?
    Ans :
  1.Map : Map is used when I want to change or transform each item in a list. It makes a new list with the changed items.
  2.Filter: Filter is used when I want to search something that starts with ‘N’. Everything that starts with ‘N’ will appear
  3.Foreach : does not return anything ,can modify original array
